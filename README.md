# Hyvelos Desktop

Hyvelos helps finance and accounts professionals turn repeated file-based work into clear desktop workflows.

Use it to read spreadsheets, clean and combine data, check exceptions, prepare reports, generate documents, handle PDFs, and keep a useful record of what ran.

Hyvelos is built for the practical work around payments, invoices, reconciliations, month-end reporting, payroll checks, audit preparation, document filing, and other recurring office processes.

## What You Can Do With Hyvelos

- Build workflows visually without maintaining fragile macros or scripts.
- Read Excel, CSV, forms, folders, and other common office inputs.
- Clean columns, filter rows, merge tables, stack files, sort records, summarize data, and run checks.
- Generate Excel, CSV, document, PDF, and file outputs.
- Review workflow setup issues before running.
- See progress, errors, output links, and run history.
- Use PDF and printing tools for everyday office tasks.

## Good Fits

Hyvelos is useful when your work looks like this:

- Every week you receive files that need the same cleanup.
- You copy data between spreadsheets to prepare payment, invoice, or reconciliation reports.
- You need repeatable checks before sending or filing outputs.
- You generate similar documents from rows in a table.
- You handle batches of PDFs, folders, or print jobs.
- You want the process to be visible enough for another finance person to understand.

## Workflow Builder

The workflow builder guides you from source data to finished results.

1. Add a source, such as Excel, CSV, a folder list, or an input form.
2. Add steps that clean, transform, compare, filter, merge, or prepare the data.
3. Add output steps that save the finished work.
4. Review setup issues before running.
5. Run the workflow and inspect progress, results, and output files.

The builder is designed for business users who understand the work but do not want to become software developers to automate it.

## Common Workflow Examples

- Prepare a payment upload file from multiple spreadsheets.
- Check invoice records before sharing them with a client or manager.
- Compare bank statements with internal records.
- Create aging reports and collection follow-up lists.
- Generate letters or forms from spreadsheet rows.
- Merge, split, compress, or organize PDF files.
- Watch a folder and print incoming files with predictable settings.

## Safety And Review

Hyvelos runs the workflow you configure. You remain responsible for checking source data, workflow setup, output destinations, and final results before using them for payments, filings, payroll, accounting records, tax records, legal documents, compliance work, or other important business actions.

Hyvelos helps reduce repetitive work and avoidable mistakes, but it does not replace professional review.
